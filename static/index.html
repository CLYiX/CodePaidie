配置 ChatGPT Custom GPT
  1. 启动本服务：python server.py
  2. 启动内网穿透：ngrok http 8000
  3. 复制 ngrok 给的 HTTPS 地址（如 https://abc123.ngrok-free.app)
  4. 回到本页"设置"，粘贴到"公网地址"栏
  5. 打开 chatgpt.com/gpts -> 创建 GPT
  6. 在 Actions 部分，点击"创建新 Action"
  7. 粘贴"OpenAPI 接口"页的 JSON
  8. 认证方式选 API Key，类型选 Custom，Header 填 X-API-Key
  9. 粘贴"设置"页的 API 密钥
  10. 保存，测试！
接口列表
GET /api/read?path=... 读取文件内容
POST /api/write 写入文件（请求体：{"path": "...", "content": "..."}）
GET /api/list?path=. 列出目录内容
GET /api/tree?path=. 获取目录树视图
GET /api/delete?path=... 删除文件或目录
GET /api/copy?source=...&dest=... 复制文件或目录
GET /api/move?source=...&dest=... 移动或重命名
GET /api/mkdir?path=... 创建目录
GET /api/info?path=... 文件信息（大小、时间等）
GET /api/search?query=... 按文件名搜索
GET /api/grep?query=... 搜索文件内容
GET /api/exec?command=... 执行命令（沙箱内）
CodePaidie 浏览器插件

v2: 给任何 AI 网页版注入本地文件操作能力。无需平台支持 Actions，装个插件就能用。

ChatGPT Gemini Kimi 通义 文心 DeepSeek
安装步骤
  1. 打开 Chrome，访问 chrome://extensions
  2. 开启右上角「开发者模式」
  3. 点击「加载已解压的扩展程序」
  4. 选择 CodePaidie 项目中的 extension/ 文件夹
  5. 点击插件图标，填入 Server URL 和 API Key
  6. 打开任意 AI 网页版，开始使用
工作原理

插件自动注入系统提示词，告诉 AI 你有本地文件操作能力。
当 AI 需要读写文件时，输出 标记。
插件拦截标记 → 调用 CodePaidie API → 把结果注入回对话。
整个过程对 AI 来说是透明的，就像它天生就有文件操作能力一样。
